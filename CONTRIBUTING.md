# Contributing to Material Passport

Thank you for your interest in contributing to Material Passport.

## Getting Started

1. Fork the repository
2. Clone your fork locally
3. Create a new feature branch
4. Make your changes
5. Commit your changes
6. Push to your fork
7. Open a Pull Request

## Development Guidelines

* Use TypeScript
* Follow the existing project structure
* Write clear and descriptive commit messages
* Keep code clean and documented

## Reporting Issues

If you find a bug or have a feature request:

1. Open a GitHub Issue
2. Describe the problem clearly
3. Include steps to reproduce if applicable

## Pull Requests

Before submitting a Pull Request:

* Ensure the project builds successfully
* Test your changes
* Update documentation if necessary

## Code of Conduct

Be respectful and constructive when interacting with other contributors.

Thank you for helping improve Material Passport.
